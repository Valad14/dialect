# Автоматическая транскрипция диалектной речи

Минималистичное Streamlit-приложение для автоматической транскрипции русской диалектной звучащей речи.

Приложение делает два результата:

1. **Орфографическая запись** — обычный текст, полученный из аудио через Speech2Text на базе Whisper.
2. **Фонетическая транскрипция** — приближенная запись произношения через speech2phoneme / speech@phonetics в двух вариантах: кириллица и IPA.

## Что изменено в этой версии

- интерфейс стал проще и понятнее для обычного пользователя;
- оставлена только одна настройка — **модель Whisper**;
- убраны технические фильтры: язык, устройство, word timestamps, подсказка Whisper, диалектные переключатели;
- добавлены вкладки **Руководство** и **О проекте**;
- добавлен файл `packages.txt` для установки `ffmpeg` на Streamlit Cloud;
- добавлен резервный механизм `imageio-ffmpeg`, который помогает избежать ошибки `[Errno 2] No such file or directory: 'ffmpeg'`;
- архив подготовлен так, чтобы `app.py`, `requirements.txt` и `packages.txt` лежали в корне проекта.

## Быстрый запуск на Streamlit Cloud

1. Распакуйте архив.
2. Создайте новый репозиторий на GitHub.
3. Загрузите **содержимое папки** в корень репозитория. В корне должны лежать:

```text
app.py
requirements.txt
packages.txt
runtime.txt
src/
.streamlit/
```

4. Откройте Streamlit Community Cloud.
5. Создайте новое приложение из GitHub-репозитория.
6. В поле **Main file path** укажите:

```text
app.py
```

7. Нажмите **Deploy**.

Важно: не кладите весь проект в дополнительную вложенную папку внутри GitHub. Файл `packages.txt` должен быть именно в корне репозитория, иначе Streamlit Cloud может не установить `ffmpeg`.

## Локальный запуск

```bash
python -m venv .venv
source .venv/bin/activate      # Windows: .\.venv\Scripts\activate
pip install -U pip
pip install -r requirements.txt
streamlit run app.py
```

Для локального запуска желательно установить системный `ffmpeg`. На Streamlit Cloud он устанавливается через `packages.txt`, а дополнительно предусмотрен резервный вариант через `imageio-ffmpeg`.

## Как пользоваться

1. Откройте приложение.
2. В боковой панели выберите модель Whisper:
   - `tiny` — самый быстрый вариант, лучше для бесплатного Streamlit Cloud;
   - `base` — баланс скорости и качества;
   - `small` — точнее, но требует больше памяти и времени.
3. Загрузите аудиофайл: WAV, MP3, FLAC, M4A, OGG или WebM.
4. Нажмите **Создать транскрипцию**.
5. Скачайте TXT или JSON, если нужно сохранить результат.

Также можно открыть блок «У меня уже есть текст» и проверить только фонетическую транскрипцию без аудио.

## Структура проекта

```text
app.py                              # Streamlit-интерфейс
requirements.txt                    # Python-зависимости
packages.txt                        # Linux-зависимости Streamlit Cloud: ffmpeg
runtime.txt                         # версия Python для деплоя
USER_GUIDE.md                       # руководство пользователя
src/dialect_transcription/
  audio.py                          # работа с временными аудиофайлами
  runtime.py                        # проверка и резервное подключение ffmpeg
  models.py                         # структуры результата
  report.py                         # экспорт TXT/JSON
  speech2text.py                    # Speech2Text через Whisper
  speech2phoneme.py                 # speech2phoneme / speech@phonetics правила
```

## Ограничения

Whisper распознает речь в орфографический текст, а фонетическая часть работает по правилам. Поэтому результат нужно проверять вручную, особенно если запись шумная, в речи несколько говорящих, есть редкие диалектные слова или важна точная постановка ударения.
