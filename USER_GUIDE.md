# Руководство пользователя

## Для чего нужна программа

Программа помогает быстро получить первичную транскрипцию диалектной речи из аудиозаписи. Она подходит для учебных проектов, демонстрации, предварительной расшифровки интервью и подготовки материала к ручной лингвистической проверке.

## Что вы получаете на выходе

- **Орфографическая запись** — обычный русский текст после Speech2Text.
- **Фонетическая транскрипция кириллицей** — удобная запись произношения для чтения.
- **IPA-транскрипция** — запись в международном фонетическом алфавите.
- **TXT-отчет** — файл для чтения и печати.
- **JSON** — структурированный файл с текстом, сегментами и примененными правилами.

## Пошаговая инструкция

1. Откройте приложение по ссылке Streamlit.
2. В боковой панели выберите модель распознавания.
3. Загрузите аудиофайл или вставьте готовый текст.
4. Нажмите кнопку запуска.
5. Проверьте результат на экране.
6. Скачайте TXT или JSON.

## Как выбрать модель

- `tiny` — быстрее всего, подходит для проверки и бесплатного Streamlit Cloud.
- `base` — хороший вариант по умолчанию.
- `small` — лучше качество, но дольше обработка.

## Если появилась ошибка ffmpeg

В обновленной версии есть два способа решения:

1. `packages.txt` устанавливает системный `ffmpeg` на Streamlit Cloud.
2. `imageio-ffmpeg` используется как резервный вариант внутри Python.

Если ошибка все равно появилась, проверьте GitHub-репозиторий: `packages.txt` должен лежать в корне рядом с `app.py`, а приложение нужно заново развернуть или пересоздать на Streamlit Cloud.

## Советы по качеству аудио

- используйте короткие записи для теста;
- избегайте сильного фонового шума;
- говорящий должен быть достаточно близко к микрофону;
- если говорят несколько человек одновременно, результат может ухудшиться;
- для важных материалов обязательно выполните ручную проверку.
